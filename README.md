# Makeup Review System

![Python](https://img.shields.io/badge/Python-blue?style=flat-square)

## 📖 Description

A complete makeup review platform where regular users can register, login, add reviews with images, view reviews, delete their own reviews, and mark reviews as helpful. Admin has full control to delete any review, view and delete users, plus an admin dashboard.
